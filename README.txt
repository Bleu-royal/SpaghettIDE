Voici le dossier principal de SpaghettIDE.
Pour lancer SpaghettIDE, vous devez exécuter le fichier "main.py".
Le dossier content contient des fichiers divers.
Le dossier gui contient tous les modules relatifs à l'IHM.
Le dossier kernel contient un fichier de variables.
Le dossier language contient le module relatif au langages ainsi que les dictionnaires de langues.
Le dossier lexerc contient les modules relatifs à l'analyse du code (ply, plyplus, et des grammaires).
Le dossier snippets contient des bouts de code (auto-complétion).
Le dossier systeme contient les modules relatifs au travail d'arrière plan.
Le dossier themes contient les modules qui permettent de changer le thème de SpaghettIDE.

Le fichier "xml.py" est le module qui gère le xml sur l'ensemble de SpaghettIDE.

Le fichier "gen_project.py" est le module qui gère la création de projet dans SpaghettIDE.

Ce travail a été réalisé par Alexis CARREAU, Thomas LÉCLUSE, Emma MAUGER et Théo SARRAZIN dans le cadre d'un travail en deuxième année de Licence Informatique à l'Université de Caen Normandie.

On retrouve sur notre logiciel toutes les fonctionnalités offertes par des concurrents, comme l'auto-compltétion, l'analyse syntaxique, la correction de fautes de frappes, un compilateur intégré et bien plus encore ! 
Découvrez par vous-même en téléchargeant le SpaghettIDE.

Notre IDE est bien évidemment gratuit mais il est surtout open source ! Vous pouvez donc voir comment il est conçu et le modifier à votre guise ! SpaghettIDE est programmé en langage Python, avec la bibliothèque graphique QT (pour C++) interprété par PySide qui fait le lien vers Python. Il vous faudra donc installer ceci auparavant :

Python version 3.4 : https://www.python.org/downloads/

QT version 4.8.7 : https://www.qt.io/download/

PySide version 1.2.4 : http://wiki.qt.io/Category:LanguageBindings::PySide::Downloads


Documentation : https://doc.qt.io
