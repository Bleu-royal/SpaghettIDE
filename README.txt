Voici le dossier principal de l’IDE permettant de l’exécuter. Nous pouvons remarquer un dossier e_g_rapports pour les rapports du groupe, un module GUI pour l’interface graphique, un dossier images pour les images (!), un module KERNEL pour les liens boutons / fonctions et la création du dossier Workplace, un dossier PLY pour LEX et YACC, enfin, nous retrouvons un module SYSTEME pour la coloration, la gestion des documents, sauvegarde, ouverture…, la gestion des fichiers et des projets. Les fichiers présents dans  le dossier principal servent d’une part pour utiliser LEX (lexer.py), il y a aussi le fichier d’éxécution main.py ainsi que des fichiers générés par LEX et YACC.
